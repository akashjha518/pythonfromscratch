# Python From Scratch

This repository contains Python practice exercises and small projects grouped by topic. It looks like a course or assignment workspace, with each numbered folder covering one subject area.

## Repository Layout

- `1.BasicPythonConcepts`
- `2.ControlStructuresInPython`
- `3.Functions&ModulesInPython`
- `4.Files_Exceptions_and_Errors_in_Python`
- `5.Data_Structures_and_Strings_in_Python`
- `6.CALCULATOR_USING_TKINTER`
- `7.Implementing_Database_Operations`
- `8.Flask_Registration_Form`
- `9.Django_REST_API`
- `10.Web_Scraping`

Only the first six folders currently contain files. The later folders exist as placeholders for future work.

## Included Work

- Basic arithmetic and greeting programs
- Control structure exercises such as even/odd checks and summation
- Functions and modules practice
- File handling and error handling examples
- Dictionary and list slicing exercises
- A Tkinter calculator GUI

## Project Archives

The root folder also includes packaged submissions:

## Running the Examples

Most scripts are standard Python files and can be run with:

```bash
python path\to\script.py
```

For the Tkinter calculator:

```bash
python 6.CALCULATOR_USING_TKINTER\calculator.py
```

## Requirements

- Python 3.x
- Tkinter for the calculator project
